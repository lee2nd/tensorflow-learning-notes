# 鮑魚年齡預測 (Abalone Age Prediction）

## 欄位特徵 (Features)
資料表顯示了鮑魚的各種物理特徵：

- Length (長度)、Diameter (直徑)、Height (高度)
- Whole weight (總重量)
- Shucked weight (去殼肉重)
- Viscera weight (內臟重量)
- Shell weight (外殼重量)

## 預測目標 (Target)
最右邊的欄位是 Age (年齡)。在實際研究中，直接計算鮑魚的年齡（需要切片並在顯微鏡下數貝殼年輪）非常費時，因此這個模型的主題是「利用鮑魚的物理外觀與重量特徵，來預測其年齡」
